About AstroStocks

AstroStocks is a unique MERN-stack-based platform that fuses stock market analysis with astrological insights. By analyzing financial trends and planetary positions, it provides users with a new perspective for making investment decisions.

🔧 Tech Stack

  • MongoDB – NoSQL database for storing prediction and astrology data
  • Express.js – Backend framework for server-side APIs
  • React.js – Frontend for dynamic user interface
  • Node.js – JavaScript runtime to run the backend

🪐 Why Astrology?

Planetary alignments and zodiac energies have been linked with behavioral and emotional shifts. AstroStocks uses these shifts to offer timing-based suggestions for your stock actions.

🚀 Our Vision

To empower investors by merging data science and ancient wisdom, offering both technical and intuitive insights for better financial choices.
